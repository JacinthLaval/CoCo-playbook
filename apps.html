Precision Advisor & ER Console

Clinical decision support with pharmacogenomics, ambient scribe, and population health

Deployed

A comprehensive clinical workstation built with React Native (Expo). Six tabs covering the patient journey: Patient profile with PGx variants and risk gauges, Ambient Scribe with real-time transcription and SOAP note generation, Population Health dashboards, ER Console with pharmacogenomic drug-gene interaction alerts, cuGraph variant similarity network, and a Platform Governance view showing live RBAC, masking, and audit status. MCP Server integration, 3 stored procedures, and 3-layer agent guardrails.

HL7v2/FHIR → OMOP/Tuva Transformer

Snowflake Native App for clinical data interoperability — Marketplace-ready

Native App V1.5

A Snowflake Native App that transforms FHIR R4 (JSON + XML) and HL7v2 pipe-delimited messages into OMOP CDM v5.4 or Tuva Input Layer tables. Auto-detects input format, runs 23 OMOP mappers or 13 Tuva mappers with OHDSI Athena vocabulary crosswalks. Streamlit UI with 7 tabs: Configure, Run (with per-mapper progress), History, Vocabulary Browser, Coverage Report, Data Quality, and Explore. Graduated testing framework (Level 1–3) with mandatory pause gate.

Ontology Stack Builder

CoCo skill — schema-to-knowledge-graph-to-agent in a single conversational session

Published

A CoCo skill that generates the full Ontology-on-Snowflake stack from any relational schema through a 7-phase gated workflow. Builds metadata tables, abstract views, semantic models, and a Cortex Agent automatically. Supports both Knowledge Graph (KG_NODE/KG_EDGE) and Direct Table paths. Includes a Streamlit-in-Snowflake visualizer with interactive graph view, hierarchy explorer, and coverage mapping. Battle-tested on OMOP CDM + FHIR healthcare data — 130K knowledge graph nodes, 37 ontology classes.
