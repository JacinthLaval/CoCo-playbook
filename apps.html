HL7v2/FHIR → OMOP/Tuva Transformer

Snowflake Native App for clinical data interoperability — Marketplace-ready

Native App V1.5

A Snowflake Native App that transforms FHIR R4 (JSON + XML) and HL7v2 pipe-delimited messages into OMOP CDM v5.4 or Tuva Input Layer tables. Auto-detects input format, runs 23 OMOP mappers or 13 Tuva mappers with OHDSI Athena vocabulary crosswalks. Streamlit UI with 7 tabs: Configure, Run (with per-mapper progress), History, Vocabulary Browser, Coverage Report, Data Quality, and Explore. Graduated testing framework (Level 1–3) with mandatory pause gate.
